API接口参数规范

base() : string
测试地址：
http://staums.zhahehe.com/xxx/xxx
线上地址：
http://staums.zhahehe.com/xxx/xxx

请求类型
            POST or GET
请求参数
          smsdo　：短信用途  必填  【注册：1，重置密码：2】
          ......

Response

string

返回JSON数据格式

JSON数据格式：
        code 返回码
        msg  提示信息
        data 数组
返回码规范
        -1 系统繁忙
        0  请求成功
        1开头的五位数   权限错误码
        4开头的五位数   参数错误码
        6开头的五位数   数据层错误码
成功：
  {
    "code": "0",
    "msg": "登陆成功！",
   "data": {
     "user_id": "2"
   }
  }
失败：
  {
    "code": "60011",
    "msg": "密码不正确！",
       "data": {}
  }

公共上传图片接口

upload() : string
测试地址：
http://staums.zhahehe.com/apiBase/upload
线上地址：
http://staums.zhahehe.com/apiBase/upload

允许上传的类型 ：'jpg', 'gif', 'png'
图片大小没有限制
宽度超过600，高度超过600时自动压缩
POST参数
          file　：文件  必填

Response

string

返回JSON数据格式

成功：
  {
    "code": "0",
    "msg": "上传成功！",
       "data": {
        "url": "http://staums.zhahehe.com/uploads/201707/10/783770112012.jpg"
      }
  }
失败：
  {
    "code": "60014",
    "msg": "上传失败，请重新上传！",
       "data": {}
  }
code：
  {
    0     =     '上传成功！'
    40011 =     '请上传文件！'
    60014 =     '上传失败，请重新上传！'
  }
