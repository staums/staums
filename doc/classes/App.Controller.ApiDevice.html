POST参数
             parent_id            ：父级ID
          name　                ：设备名称 必填
          soft                ：排序
          room_id            ：房间ID
          port_id             ：接口ID
          device_type_id     ：设备类型 必填
          device_model_id    ：设备模板
          icon_id             ：图标ID
          pic                 ：图标地址
          is_lock             ：是否验证：1否2是

string
